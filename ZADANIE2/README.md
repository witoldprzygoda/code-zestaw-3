# Zadanie 2

Napisz klasę o nazwie BigInt, która będzie przechowywać dowolnie wielką liczbę (całkowitą, a zatem może być ujemna) w postaci typu std::string oraz pozwalać dodawać i odejmować (przeciążone operatory + i -) oraz pozwalać na wypisanie wartości na ekran za pomocą operatora <<. Konstruktor oraz operator= oczywiście powinien zostać napisany. Tworząc obiekt BigInt podajemy jako argument liczbę w postaci łańcucha znakowego. Dodatkowo przeciąż jednoargumentowe operatory + i -.

Pliki bigint.h jest częściowo wypełniony, wskazując kierunek dalszego rozwoju kodu (definicje umieszczać w bigint.cpp). Plik main.cpp zawiera krótki program demonstrujący. Poza tym obecne są również testy, za pomocą których (z użyciem CMake) można sprawdzić poprawność kodu. To zadanie jest za **2 pkt**.
