# Zadanie 3

Napisz program do mnożenia macierzy o zadanych rozmiarach. Macierz Am×n oraz Bn×p daje macierz Cm×p. Macierze wczytać z pliku ASCII, zapisując w postaci wierszy i kolumn oddzielonych spacjami (np. macierz A będzie mieć m wierszy i n kolumn). Po przemnożeniu wynik zapisać do jakiegoś pliku. Więcej informacji oraz przykładowy program: https://eduinf.waw.pl/inf/alg/001_search/0074.php

W pliku macierze.h są deklaracje spodziewanych definicji funkcji (do napisania w pliku macierze.cpp). Nazwa Macierz oznacza zagnieżdżony std::vector<std::vector<double>>. Spodziewany format danych wejściowych można odgadnąć z programu main.cpp, widać, że pierwsze dwie liczby w pliku to rozmiary macierzy. Zadaniu towarzyszą testy (Google Test do uruchomienia poprzez CMake).
